# Zu’s Portfolio

> 「Nuxt.js + Atomic Desing + Netlify」を使用したポートフォリオサイト

## URL
- https://zsp2088dev.netlify.com/

## 動作環境
- Node.js v10.16.3
- Nuxt.js v2.10.2

## ビルドとセットアップ

``` bash
# パッケージのインストール
$ yarn install

# 開発環境で実行 (localhost:3000)
$ yarn dev

# ビルドして実行
$ yarn build
$ yarn start

# アプリケーションをビルドして静的ファイルを生成
$ yarn generate
```
